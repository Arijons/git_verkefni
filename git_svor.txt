Git spurningar og svör

1. Hvað gera eftirfarandi Linux skipanir?
	cd
	cd ~
	ls
	pwd
	mkdir

	cd: skiptir um möppu
	cd ~: fer yfir í  heima möppuna
	ls: listar skrárnar í möppunni
	pwd: sýnir í hvaða directoryum maður er 
	mkdir: býr til directory

2. Hvað gera eftirfarandi git skipanir og hvers vegna eru þær gagnlegar?
	git clone
	git log
	git status
	git diff
	git checkout
git clone: býr til nýtt repocetory sem er alveg eins og óháð.

git log: til að sjá  allar skráningar.

git status: hún sínir stöðuna á addinu, commitinu og greinunum.

git diff: sýnir breytingarnar sem eru í næstu skráningu.

git checkout: til þess að skoða einstækar útgáfur.

3. Hver er munurinn á eftirfarandi git skipunum:
	a) git diff 
	b) git diff --staged 
	c) git diff commit1 commit2
a: það sem er búið að commintera.

b: það sem er búið að adda.

c: það er munurinn á commit1 og commit2.

4. 	Hvað er útgáfustýring (e.version control)?
það er stýring á mismunandi útgáfum ,þegar maður er að bæta eða breyta verkefnum eða skjölum sínum.

 5.	Hverjir eru helstu kostir við að nota GIT?
 til að hafa útgáfu á stýringu.

 6.	Hversu oft telur þú að eigi að gera commit í verkefni, rökstuddu?
 eftir stórar og markverðar breytingar á forriti eða verkefni.

 7.	Hvað er "Working directory"?" "Staging area"?" og "Repository" í GIT?

Working directory: skjalið með nýjustu breytingunum.

Staging area: skjalið með síðustu breytingunum sem var adda.

Repositry: verkefna geymslan sem geymit allar breytingar og allar útgáfur.

8.	Hvenær er sniðugt að nota greinar (branches)? 
þegar maður prófar eitthvað nýtt og vill ekki skemma það sem maður er með.
